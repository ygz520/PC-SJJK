所属分类

工商信息

请求方式

GET

接口描述

查询企业的基本工商信息，包括企业名称、统一社会信用代码、法定代表人、注册资本、成立日期、经营状态等。

接口地址

https://api.bidizhaobiao.com/v1/enterprise/baseInfo
参数名 类型 必填 说明
keyword string 是 企业名称、注册号或统一社会信用代码
page int 否 页码，默认1
pageSize int 否 每页条数，默认20，最大100
参数名 类型 说明
code int 状态码，200表示成功
message string 返回信息
data object 企业工商信息数据
data.name string 企业名称
data.creditCode string 统一社会信用代码
data.legalPerson string 法定代表人

请求示例

GET https://api.bidizhaobiao.com/v1/enterprise/baseInfo?keyword=北京比地数据科技有限公司

响应示例

{ "code": 200, "message": "success", "data": { "name": "北京比地数据科技有限公司", "creditCode": "91110108MA00123456", "legalPerson": "张三", "registeredCapital": "1000万元", "establishDate": "2015-06-18", "status": "存续" } }
